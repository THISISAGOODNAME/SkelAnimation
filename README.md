An implement for [***C++ Game Animation Programming***](https://www.packtpub.com/product/hands-on-c-game-animation-programming/9781800208087)

# Features

- Mesh Skinning
  - [x] CPU Skinning
  - [x] GPU Skinning
- Blending
  - [x] Animation Blending
  - [x] Animation Crossfading
  - [x] Additive Blending
- IK
  - [x] CCD IK
  - [x] FABR IK
  - [x] Foot IK demo
- Others
  - [x] DQS
  - [x] GPU Skinning Instancing
